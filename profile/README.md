The [SaruMC Foundation](https://saru.gg) (ASF) is home to more
than [20 software projects](https://projects.saru.gg), many of which
host their code repositories in this GitHub org.

Software in this org is released under the [Apache
License](https://www.saru.gg/licenses/LICENSE-2.0).

ASF projects follow the ASF [vulnerability handling 
process](https://saru.gg/security/#vulnerability-handling).
